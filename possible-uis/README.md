These are throwaway M4 calendar-aesthetic explorations for Javi's Journal, focused only on calendar chrome with no stamps, stickers, photos, or journal content. View them by opening each `.html` file directly in a browser, or by dropping the files onto a phone for offline viewing: `ui-1-paper.html` explores paper/ink, `ui-2-pastel.html` explores soft cozy pastel, and `ui-3-scrapbook.html` explores scrapbook/craft. The chosen direction will be rebuilt as real React + CSS Modules during M4.
